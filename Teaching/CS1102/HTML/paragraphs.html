HTML Feature to Explore

Add a title of the webpage

The page title can 1) help you quickly identify a webpage on a browser if you open many tabs; 2) be used by search engine to determine the relevance with respect to keyword search; 3) be used as the default title when you bookmark your webpage.

The page title is added between the <titled> </title> tags in the HTML file.
